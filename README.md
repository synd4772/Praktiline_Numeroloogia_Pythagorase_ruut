Pythagorase kvadraat (tabel, Pythagorase horoskoop) on üks vanakreeka filosoofi ja matemaatiku Pythagorase väljatöötatud meetodeid inimese sünniaja analüüsimiseks, mis ühendas araablaste, druiidide, foiniiklaste ja egiptlaste matemaatilised süsteemid inimese loomusteadustega. Kõik inimesed, kes on siia ilma sündinud, saavad oma numbri vibratsiooni, mis kannab teatud omadusi. Maatriks võimaldab sünnikuupäeva järgi kindlaks teha tüüpilised iseloomujooned, mis on inimesele sünniga kaasa antud.
